# 📱 Инструкция: Сборка Backwoods без Mac

Тебе НЕ нужен Mac и Xcode. Всё собирается через GitHub Actions
(GitHub даёт бесплатный macOS сервер на 2000 минут в месяц).

---

## Что нужно

| Что | Где взять |
|-----|-----------|
| Git | https://git-scm.com/download/win |
| GitHub аккаунт | https://github.com/join |
| VPS в Нидерландах | Любой: Hetzner, DigitalOcean, Vultr (~€4/мес) |
| Apple Developer (опционально) | https://developer.apple.com ($99/год) |
| TrollStore на iPhone (если без Apple Developer) | Зависит от версии iOS |

## Шаг 1: Подготовка на VPS (настройка VPN сервера)

Берёшь VPS с Ubuntu 22.04 в Нидерландах. Подключаешься через SSH.

```bash
# Загрузи скрипт на сервер
scp server/setup-wireguard-server.sh root@АДРЕС_VPS:~/

# Подключись к серверу
ssh root@АДРЕС_VPS

# Запусти скрипт
chmod +x setup-wireguard-server.sh
sudo ./setup-wireguard-server.sh
```

Скрипт сам:
- Установит WireGuard
- Сгенерирует ключи сервера
- Создаст 30 конфигураций клиентов
- Включит файрвол
- Запустит WireGuard

После завершения, скопируй конфигурацию для первого клиента:

```bash
cat /etc/wireguard/peers/json/peer-01.json
```

**Сохрани этот JSON** — он понадобится на Шаге 4.

## Шаг 2: Подготовка репозитория на Windows

Открой PowerShell и выполни:

```powershell
# Перейди в папку проекта
cd C:\Users\Administrator\Desktop\backwoods

# Запусти скрипт подготовки
.\scripts\setup-repo.ps1
```

Скрипт:
- Создаст Git репозиторий
- Добавит .gitignore
- Проверит все файлы
- Сделает первый коммит

## Шаг 3: Создание GitHub репозитория

1. Открой https://github.com/new
2. Имя: `backwoods`
3. Видимость: **Private** (приватный!)
4. НЕ ставь галочки Initialize (README, gitignore, license)
5. Нажми **Create repository**

Затем в PowerShell:

```powershell
# Замени YOUR_USERNAME на своё имя GitHub
git remote add origin https://github.com/YOUR_USERNAME/backwoods.git
git push -u origin main
```

## Шаг 4: Добавление VPN конфигурации (секрет)

1. Открой репозиторий на GitHub
2. **Settings** → **Secrets and variables** → **Actions**
3. Нажми **New repository secret**
4. Имя: `TUNNEL_CONFIG_JSON`
5. Значение: вставь JSON из Шага 1 (содержимое `peer-01.json`)
6. Нажми **Add secret**

Пример значения:
```json
{
    "type": "wireGuard",
    "wireGuard": {
        "interface": {
            "privateKey": "aBcDeFgHiJkLmNoPqRsTuVwXyZ1234567890aBcD=",
            "addresses": ["10.0.0.2/32"],
            "dns": ["1.1.1.1", "1.0.0.1"],
            "mtu": 1280
        },
        "peer": {
            "publicKey": "xYzAbCdEfGhIjKlMnOpQrStUvWxYz1234567890=",
            "presharedKey": "pReShArEdKeY1234567890aBcDeFgHiJkLmNoPqRs=",
            "endpoint": "123.45.67.89:51820",
            "allowedIPs": ["0.0.0.0/0", "::/0"],
            "persistentKeepalive": 25
        }
    }
}
```

## Шаг 5: Запуск сборки

1. Открой репозиторий на GitHub
2. Вкладка **Actions**
3. Слева: **🔨 Build Backwoods IPA**
4. Справа: **Run workflow**
5. Параметры:
   - `telegram_ref`: оставь `master` (или укажи конкретный тег Telegram)
   - `skip_tests`: можно поставить ✓ для ускорения первой сборки
   - `unsigned_build`: поставь ✓ (для TrollStore)
6. Нажми **Run workflow** (зелёная кнопка)

⏱ Сборка займёт ~60-120 минут (первый раз дольше из-за кэша).

---

## Шаг 6: Скачивание IPA

1. Когда сборка завершится (зелёная галочка ✅):
2. Нажми на завершённый workflow run
3. Внизу раздел **Artifacts**
4. Скачай **Backwoods-xxxxx** (это ZIP с IPA внутри)
5. Разархивируй → получишь `Backwoods.ipa`

## Шаг 7: Установка на iPhone

### Вариант А: TrollStore (без Apple Developer)

1. Убедись, что TrollStore установлен на iPhone
2. Перенеси `Backwoods.ipa` на iPhone (AirDrop, файлы, USB)
3. Открой IPA через TrollStore
4. TrollStore установит приложение

### Вариант Б: AltStore (с Apple Developer $99/год)

1. Установи AltServer на Windows: https://altstore.io
2. Подключи iPhone по USB
3. Установи AltStore на iPhone через AltServer
4. Перенеси IPA на iPhone
5. Открой IPA через AltStore

### Вариант В: Sideloadly (без Apple Developer, обновлять каждые 7 дней)

1. Скачай Sideloadly: https://sideloadly.io
2. Подключи iPhone по USB
3. Перетащи `Backwoods.ipa` в Sideloadly
4. Введи Apple ID (обычный, бесплатный)
5. Нажми Start

⚠️ Бесплатный Apple ID = перепподписывать каждые 7 дней.
Платный ($99/год) = работает год.

## Добавление нового пользователя

На VPN сервере уже создано 30 конфигураций.
Для каждого нового пользователя:

1. SSH на сервер
2. `cat /etc/wireguard/peers/json/peer-02.json` (или peer-03, и т.д.)
3. Соберите отдельную IPA с этим конфигом:
   - Создайте новый секрет `TUNNEL_CONFIG_JSON` с новым peer JSON
   - Запустите сборку
   - Скачайте IPA

Или: все пользователи с одним peer — но тогда одновременно будет
работать только один (WireGuard не любит дубли).

## Решение проблем

### Сборка упала с ошибкой

1. Откройте упавший workflow run
2. Нажмите на failed step
3. Прочитайте лог ошибки
4. Скачайте **build-logs-xxx** из Artifacts

### "WireGuardKit.framework не найден"

Wireguard-apple мог обновить структуру проекта.
Проверь `wireguard-apple/WireGuard.xcodeproj` — возможно схема
называется по-другому.

### "Telegram.app не найден"

Telegram мог обновить билд-систему.
Проверь `telegram-ios/build-system/Make/Make.py` — возможно
изменились параметры.

### GitHub Actions: лимит минут

Бесплатный GitHub аккаунт: 2000 минут/месяц macOS.
Одна сборка ≈ 60-120 минут. Хватит на ~15-30 сборок в месяц.

## Безопасность

⚠️ **ВАЖНО:**

- Репозиторий ОБЯЗАТЕЛЬНО приватный (конфигурации содержат ключи)
- НЕ коммитьте реальные ключи WireGuard в код
- Используйте GitHub Secrets для конфигурации
- Серверные ключи храните только на сервере
- Каждому пользователю — свой peer (уникальные ключи)
