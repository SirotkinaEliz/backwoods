Установка через SideStore

Переподписка происходит автоматически каждые 7 дней — делать ничего не нужно

1
Установи SideStore
Открой в Safari → sidestore.io → нажми Install
После установки: Настройки → Основные → VPN и управление устройством → Доверять
2
Создай Pairing File (один раз)
Открой SideStore → при первом запуске нажми Generate Pairing File и следуй инструкции на экране
3
Добавь Source GLUSH
В SideStore → Browse → Sources → кнопка + → вставь адрес:
https://sirotkinaeliz.github.io/backwoods/apps.json
4
Установи GLUSH
В списке найди GLUSH → нажми Free → Install → жди ~1 минуту
✓
Запусти GLUSH
При первом запуске нажми Разрешить на запрос VPN — готово, Telegram работает через зашифрованный туннель 🔒

Частые вопросы

SideStore говорит "App expired"?
Открой SideStore → нажми Refresh All
Не появляется запрос VPN?
Настройки → Основные → VPN и управление устройством → GLUSH → Доверять → открой заново
iOS 26 — SideStore не работает?
SideStore обновляется под новые iOS за 1–2 недели. Следи на sidestore.io
⚠️ Важно: Для работы WireGuard VPN приложение запросит разрешение на создание VPN-туннеля. Разрешите его при первом запуске.
💡 Переподписка: С бесплатным Apple ID приложение работает 7 дней. SideStore переподписывает автоматически. Для остальных — повторите установку через 7 дней.
