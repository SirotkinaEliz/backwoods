SideStore Рекомендуется

Без компьютера · iOS 16+ · Работает на iOS 18 и 26
  1. Установите SideStore через sidestore.io
  2. В SideStore откройте вкладку Browse → Sources
  3. Нажмите + и вставьте URL source выше
  4. Найдите GLUSH → нажмите Install
  5. SideStore автоматически переподписывает приложение каждые 7 дней

AltStore

Нужен Mac/PC 1 раз в 7 дней · iOS 14+
  1. Установите AltServer на компьютер и AltStore на iPhone с altstore.io
  2. В AltStore откройте вкладку Browse → Sources
  3. Нажмите + и вставьте URL source выше
  4. Найдите GLUSH → нажмите Install

Feather

Без компьютера · iOS 16+ · Нужен сертификат
  1. Установите Feather
  2. Скачайте GLUSH.ipa по ссылке выше
  3. В Feather добавьте IPA → подпишите → установите

Sideloadly / AltDeploy

Нужен компьютер · iOS 14+ · Apple ID бесплатно
  1. Скачайте GLUSH.ipa на компьютер
  2. Откройте Sideloadly, введите Apple ID (бесплатный)
  3. Перетащите IPA → нажмите Start
  4. На iPhone: Настройки → Основные → VPN и управление устройством → доверяйте
⚠️ Важно: Для работы WireGuard VPN приложение запросит разрешение на создание VPN-туннеля. Разрешите его при первом запуске.
💡 Переподписка: С бесплатным Apple ID приложение работает 7 дней. SideStore переподписывает автоматически. Для остальных — повторите установку через 7 дней.
