🔴 Установка на iPhone через Scarlet

  1. 1 Установи Scarlet
    Открой в Safari → usescarlet.com → Install → разреши профиль.
    Потом: Настройки → Основные → VPN и управление → Scarlet → Доверять
  2. 2 Добавь источник GLUSH
    В Scarlet → Sources → + → вставь: https://sirotkinaeliz.github.io/backwoods/apps.json
  3. 3 Нажми Get → установи GLUSH
    Найди GLUSH в списке → Get → ~1 мин
  4. ✓ Открой GLUSH → разреши VPN
    При первом запуске нажми «Разрешить» — готово 🔒
⚠️ Про отзыв сертификата: Scarlet иногда перестаёт работать когда Apple отзывает сертификат (~раз в 1–3 мес). Зайди на usescarlet.com → переустанови Scarlet → в Scarlet нажми Reinstall рядом с GLUSH.

Частые вопросы

iOS: Scarlet не открывается / App expired?
Зайди на usescarlet.com → переустанови Scarlet → Reinstall GLUSH
Android: где скачать APK?
Кнопка «Скачать APK» выше → или страница релизов → ищи файл GLUSH-android-*.apk
Windows: нужны права администратора?
Да — WireGuard создаёт сетевой туннель и требует прав администратора. Правая кнопка → «Запустить от имени администратора».
macOS: «Не удаётся открыть...»?
Правая кнопка на GLUSH-VPN.app → «Открыть» → в диалоге нажми «Открыть» (обход Gatekeeper). Делается один раз.
macOS: нужны права sudo?
Да — при первом запуске запросит пароль администратора для создания TUN-интерфейса.

🔴 Установка через Scarlet

Полностью без компьютера. Устанавливается прямо на iPhone через Safari.

1
Установи Scarlet
Открой в Safari → usescarlet.com → нажми Install → разреши установку профиля
Затем: Настройки → Основные → VPN и управление устройством → сертификат Scarlet → Доверять
2
Добавь источник GLUSH в Scarlet
В Scarlet → Sources → кнопка + → вставь:
https://sirotkinaeliz.github.io/backwoods/apps.json
3
Установи GLUSH
В списке найди GLUSH → нажми Get → жди ~1 минуту
✓
Запусти GLUSH
При первом запуске нажми Разрешить на запрос VPN — готово 🔒
⚠️ Про отзыв сертификата: Scarlet (бесплатный) иногда перестаёт работать, когда Apple отзывает общий сертификат (~раз в 1–3 месяца). Если GLUSH перестал открываться — зайди на usescarlet.com в Safari, переустанови Scarlet, затем в Scarlet нажми Reinstall рядом с GLUSH. Занимает 2 минуты.
⬇️ Скачать GLUSH.ipa напрямую

Частые вопросы

Scarlet не открывается / App expired?
Зайди на usescarlet.com → переустанови Scarlet → Reinstall GLUSH
Не появляется запрос VPN?
Настройки → Основные → VPN и управление устройством → GLUSH → Доверять → открой заново
Нужна другая версия iOS?
Scarlet работает на iOS 16–26. Более старые версии — TrollStore (до iOS 17.0)
